# Avatar styles — generated reference

> Generated by `scripts/build-avatar-gallery.mjs` from `scripts/avatars-core.mjs`. Do not edit by hand.
> For the "when to use" judgment and the animation finding, see [index.md](index.md) and [animation.md](animation.md).

**45 offline styles**, all deterministic (`<style>:<seed>` → the same avatar every time) and all **static** — none animate natively. Grab any one:

```bash
node scripts/avatar.mjs <style>:<seed> [--size N] [--format svg|datauri|png|html] [--out FILE]
```
