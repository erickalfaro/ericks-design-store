Animated avatars · mini demo

No avatar generator animates natively — so, exactly like Lucide Animated wraps static Lucide icons in motion, you generate a static avatar and animate the wrapper. Each card is a real avatar from scripts/avatar.mjs + a CSS pattern you can lift. ← avatar gallery · animation.md · React/Motion version: vendor/animated-avatars/.

Enter — pop & fade

lorelei

When an avatar first appears — lists loading in, a new comment, a just-generated identity.

plays on mount
@keyframes pop{0%{transform:scale(.5);opacity:0}
  60%{transform:scale(1.06)}100%{transform:scale(1)}}
.avatar{animation:pop .5s cubic-bezier(.2,.8,.25,1) both}

Idle — float

bigheads

A little life on an otherwise static profile or hero. Always-on, subtle, low cost.

@keyframes float{0%,100%{transform:translateY(0)}
  50%{transform:translateY(-8px)}}
.avatar{animation:float 3s ease-in-out infinite}

Hover — wiggle

bottts

Micro-interaction on an interactive affordance (the Lucide Animated signature). Hover the robot →

↑ hover the avatar
@keyframes wiggle{15%{transform:rotate(-8deg) scale(1.04)}
  45%{transform:rotate(-5deg)}80%{transform:rotate(-1deg)}100%{transform:none}}
.avatar:hover{animation:wiggle .65s ease-in-out}

Generating — seed shuffle

pixel-art

A "rolling identity" while generating or randomizing. Leans on generation being instant & deterministic.

cycles seeds, then settles
// generation is instant → swap the avatar on a timer, decelerate, settle
let i=0, d=60; (function tick(){ show(frames[i++ % frames.length]);
  if((d*=1.14) < 320) setTimeout(tick, d); })();

Status — presence pulse

notionists

Online / live / recording state. Pure CSS ring around a static avatar — no JS.

.ring::after{content:"";/* green dot, bottom-right */
  animation:pulse 1.8s ease-out infinite}
@keyframes pulse{0%{box-shadow:0 0 0 0 rgba(63,185,80,.55)}
  70%,100%{box-shadow:0 0 0 13px rgba(63,185,80,0)}}
These 5 are patterns, not a limit. Any CSS/JS animation works on a generated avatar because it's just SVG. Respect prefers-reduced-motion (this page does). For React apps, use the Motion component in vendor/animated-avatars/AnimatedAvatar.tsx.
